# **App Name**: Kicket

## Core Features:

- Dashboard: Overview of sales, revenue, and upcoming events for organizers.
- Event Management: Create, edit, clone, and manage draft events.
- Seat Management: Design seat layouts, define ticket categories (VIP, Regular), and set pricing.
- Order Management: View all orders, process refund requests, and generate attendee lists with optional CSV export.
- Earnings: Track earnings, view commission breakdowns, request payouts, and see payment history.
- Promotions: Create promo codes and set up discount rules to boost ticket sales.
- Reports: Generate daily sales, ticket type sales, and attendance reports.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) to convey professionalism and trust.
- Background color: Light gray (#ECEFF1) to create a clean and organized feel.
- Accent color: Orange (#FF9800) to highlight important actions and information.
- Body and headline font: 'Inter' sans-serif for a modern and readable design.
- Use flat, line-based icons to maintain a clean and consistent aesthetic.
- Dashboard layout featuring clearly defined sections for sales, events, seating, and reports.
- Subtle transitions and animations on data updates to enhance user experience.
